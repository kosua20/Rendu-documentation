void LightProbe::decode ( const KeyValues &  params,
Storage  options

Setup the probe parameters from a list of key-value tuples. The following representations are possible: either: 

probe: texturetype: ...

for a static environment map or 

probe: position: X,Y,Z

for a probe renderered on the fly at the given location
