Detailed Description

Represent a rendering target, of any size, format and type, backed by a GPU framebuffer.

Framebuffer can use different shapes: 2D, cubemap, 2D array, cubemap array, but you can only render to one 2D layer at a time. For cubemaps and arrays you can select the layer when binding. For mipmapped framebuffers, you also select the mip level.
