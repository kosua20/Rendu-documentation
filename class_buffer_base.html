Detailed Description

General purpose GPU/CPU buffer, without a CPU backing store. This allows the class to be template-free (to avoid exposing GLUtilities/GPUObjects details in headers), while preserving the "pass general Object to GLUtilities for setup/upload & GPUObject only stores ID and enums" approach followed by Texture and Mesh. In practice you will want to use Buffer to benefit from CPU storage and simplified upload.

Inherited by Buffer< T >, and Buffer< ForwardLight::GPULight >.
